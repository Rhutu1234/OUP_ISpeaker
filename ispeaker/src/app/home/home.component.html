What is the iWriter? The Oxford iWriter is an interactive tool designed to help you write more effectively in English. It provides models of 19 different types of writing and takes you through the key features of each one. It provides outline structures or frameworks for all the different types of writing, into which you can add your own content. For comparison and argument essays there are extra frameworks which allow you to structure your own essay in a slightly different way from the model essays. The iWriter helps you to plan, choose vocabulary, write, and check your own written work.
